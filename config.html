配置

GitHub Token

使用 GitHub Issues 作为数据源需要配置 Personal Access Token。 Token 仅存储在您的浏览器本地，不会上传到任何服务器。

  1. 访问 GitHub Settings → Developer settings → Personal access tokens → Tokens (classic)
  2. 点击 "Generate new token (classic)"
  3. 设置 Token 名称（如：简博客）
  4. 勾选 "repo" 权限（完整仓库访问）
  5. 点击 "Generate token" 并复制生成的 Token
  6. 将 Token 粘贴到下方输入框并保存
